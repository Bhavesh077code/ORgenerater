# QR Code Generator

A simple **QR Code Generator** project built with **Node.js (Express)** for the backend and **React** for the frontend.  
It allows users to generate QR codes for any URL and display them directly in the browser.

Features
1. Generate QR codes from any link
2. Base64 QR image returned from backend
3. Responsive frontend with Tailwind CSS
4. Loading and error handling
5. Easy to extend and integrate
6. CORS enabled for frontend-backend communication

# Backend  Technologies Used
- Backend:** Node.js, Express, QRCode, CORS
- Frontend:** React, Tailwind CSS
- Other Tools:** Nodemon (for dev), Postman (for API testing)
